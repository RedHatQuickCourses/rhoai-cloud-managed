Amazon SageMaker Hands On

Lab Exercise: Get Started with Amazon SageMaker

Demo Platform Lab Environment

Launch an AWS Blank Open Environment to follow along with this guided lab.

AWS Blank Open Environment

This lab environment can take up to 20 minutes to become available, however it’s usually available in a few minutes.

Logon to the AWS Console and open SageMaker

Figure 1. Animated - Open SageMaker in the AWS Console.

  1. Use the link and credentials provided by Demo Hub to logon to the AWS Console.

  2. Use the search bar to search for sagemaker.

  3. Under service click the Amazon SageMaker tile to open Amazon SageMaker.

  4. Explore the SageMaker home page.

Create a SageMaker domain

  1. Select the SageMaker Studio menu option

  2. Select the Create a SageMaker domain option

  3. Choose the Set up for single user (Quick setup) option on the left.

  4. Click on the Set up button.

At which point an error message will appear: There is no VPC with available Subnets in this region. Please Create one.

Figure 2. Animated - Create a SageMaker Domain

Create a VPC to continue

  1. Select the Create VPC option in the error message.

  2. At top right, select Create VPC.

  3. Enter the option name tag: sagemaker_domain_demo

  4. For IPv4 CIDR enter: 172.17.0.0/16

  5. Scroll to the bottom and select Create VPC.

Figure 3. Animated - Create an AWS Virtual Private Cloud

Create a Subnet in the VPC

  1. Under the Virtual privage cloud menu, select Subnets

  2. At top right, select Create subnet.

  3. Select a VPC ID from the dropdown, use the name if multiple vpc’s exist (sagemaker_domain_demo).

  4. In the Subnet Settings, enter a Subnet name: sagemaker-demo-subnet.

  5. For IPv4 subnet CIDR block enter: 172.17.0.1/24

  6. Scroll to the bottom and select Create subnet.

Figure 4. Animated - Create an VPC Subnet

Create a Subnet in the VPC

  1. Return the SageMaker Domain setup page

  2. Select the Set up options again.

  3. You’ll recieve an Preparing SageMaker Domain Banner across the page

    1. The domain setup takes ~5-7 minutes.

  4. Once the setup completes the blue banner will disappears and the status will change to ready.

  5. Use the Amazon SageMaker link to return to the Sagemaker dashboard.

  6. Select the Studio link to refresh the dashboard.

  7. From the Get Started box select Open Studio.

You are now ready to use SageMaker Studio

Figure 5. Animated - Set up an SageMaker Domain
